     # Агенти DevDigest (.claude/agents/)

     Короткий вступний абзац — навіщо ці агенти і як вони взаємодіють між собою.

     ## researcher
     - Призначення
     - Модель / інструменти
     - Коли використовувати / коли НЕ використовувати
     - Виведення (structured report)

     ## planner
     - Призначення
     - Модель / інструменти / скили
     - Метод планування (хвилі)
     - Виведення (секції плану)

     ## implementer
     - Призначення
     - Модель / інструменти / скили / ізоляція (worktree)
     - Обмеження (тільки своя область, не Agent)
     - Виведення (звіт з diff-ом)

     ## Типовий порядок роботи (flow-діаграма ASCII)
     researcher → planner → implementer(s)

     ## Примітки
     - Посилання на AGENTS.md як основне джерело конфігурації проекту
     - Посилання на CLAUDE.md для правил сесії
