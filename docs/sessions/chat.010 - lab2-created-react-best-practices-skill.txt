     Plan: Install Skill Creator from anthropics/skills

     Context

     The project already has manually written skills
     (.claude/skills/engineering-insights/,
     .claude/skills/react-best-practices/). The user wants to replace the ad-hoc
     approach with the official Anthropic Skill Creator — a meta-skill that
     provides a structured workflow for creating, testing, evaluating, and
     iteratively improving skills. Source:
     https://github.com/anthropics/skills.git

     What Skill Creator provides

     - SKILL.md — 486-line instruction set covering the full skill creation
     lifecycle
     - agents/ — grader, comparator, analyzer subagents for automated evaluation
     - scripts/ — Python scripts for running evals, benchmarking, packaging,
     description optimization
     - eval-viewer/ — interactive HTML viewer for reviewing A/B test results
     - references/schemas.md — JSON schema for eval/benchmark files

     Claude Code auto-discovers skills from .claude/skills/<name>/SKILL.md — no
     registration needed.

     Installation steps

     1. Sparse-clone only skills/skill-creator/ from the repo

     git clone --depth=1 --filter=blob:none --sparse \
       https://github.com/anthropics/skills.git /tmp/anthropics-skills

     cd /tmp/anthropics-skills
     git sparse-checkout set skills/skill-creator

     2. Copy to project .claude/skills/

     cp -r /tmp/anthropics-skills/skills/skill-creator \
       /Users/artem.slv/Projects/goit-agentic-engineering/.claude/skills/skill-c
     reator

     3. Clean up temp clone

     rm -rf /tmp/anthropics-skills

     Result
