  'INSIGHTS\\.md' || echo '{\"systemMessage\": \"[ INSIGHTS ] Session ended — 
  did you update INSIGHTS.md?\"}'"
        }
      ]
    }
  ]

  Коли спрацьовує: коли Claude завершує відповідь і зупиняється (кінець turn).

  Що робить:
  1. git diff --name-only HEAD — перевіряє, які файли були змінені в цій сесії
  2. grep -q 'INSIGHTS\.md' — шукає серед них INSIGHTS.md
  3. Якщо INSIGHTS.md не знайдено (|| — "або") — повертає systemMessage
  4. Claude Code вбудовує це повідомлення в контекст як системне — Claude його
  "бачить" і нагадує тобі

  Ефект: якщо ти зробив нетривіальну роботу і не оновив INSIGHTS.md, Claude
  нагадає про це наприкінці сесії. Це підтримує Session protocol із CLAUDE.md.

  ---
  UserPromptSubmit хук (теж є)

  Спрацьовує на кожен твій промпт. Якщо ти пишеш /export — перехоплює команду і
  підказує ім'я файлу docs/sessions/chat.NNN.md з автоінкрементом, замість того
  щоб Claude виконував /export вручну.

  ---
  Формат відповіді хука

  Хук може повернути JSON:
  - {"systemMessage": "..."} — вставляє повідомлення в контекст Claude
  - {"continue": false} — блокує дію (використовується в UserPromptSubmit для
  /export)
  - Нічого — просто виконується без впливу на Claude
