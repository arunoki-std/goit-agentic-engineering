## What belongs here

- Gotchas with Fastify 5, Drizzle ORM, pgvector, or testcontainers discovered during work
- Non-obvious behaviors of the DI container or adapter wiring

- Surprises about the review pipeline (run-executor, SSE streaming, grounding)

## What does NOT belong here

- Things derivable from reading the code or README
- Standard Fastify/Drizzle/Postgres practices
- In-progress or volatile state
- Cross-module findings (add those to the root insights/INSIGHTS.md)
