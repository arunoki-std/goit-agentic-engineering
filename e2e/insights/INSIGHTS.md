# e2e/ insights

Findings specific to `@devdigest/e2e` (agent-browser flows).

---

## What belongs here

- Gotchas with agent-browser CLI commands, locators, or timing discovered during work
- Non-obvious behaviors of the hermetic runner or Docker networking
- Decisions about flow structure or assertion patterns with context
- Surprises about seeded data assumptions or flow ordering

## What does NOT belong here

- Things derivable from reading the code or README
- Standard browser testing practices
- In-progress or volatile state
- Cross-module findings (add those to the root insights/INSIGHTS.md)
